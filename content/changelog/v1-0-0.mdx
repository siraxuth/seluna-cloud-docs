![Seluna Storefront v1.0.0](https://images.unsplash.com/photo-1556742049-0cfed4f6a45d?q=80&w=2940&auto=format&fit=crop)

- **Multi-Tenant Storefront** รองรับ subdomain routing และ custom domain ต่อร้านได้เลย
- **Payment หลายช่องทาง** PromptPay QR, Bank Transfer, TrueMoney, Beam, PaySo, และ Wallet
- **Reseller System** ครบ — API keys, tier pricing, wallet, webhook
- **Shipping Integration** เชื่อม iShip/Pranakorn รองรับ Kerry, Flash, J&T, ThaiPost, DHL และอีกหลายเจ้า
- **LINE Bot** สั่งซื้อ ดูยอด เติมเงิน สแกนสลิป ผ่าน LINE ได้เลย
- **Digital Products** ระบบส่ง account เกม/บริการดิจิทัลแบบ encrypted อัตโนมัติ
