# การบริหารจัดการบัญชีส่วนบุคคล (Account Management)

หัวใจสำคัญของการใช้งาน Seluna Cloud คือการที่ท่านสามารถควบคุมและปรับแต่งการใช้งานให้สอดรับกับความต้องการในฐานะผู้บริโภคหรือผู้ค้าปลีกได้อย่างสมบูรณ์แบบ ผ่านส่วนหน้าบัญชีผู้ใช้งาน (Member Dashboard) ที่มีประสิทธิภาพ

---

## ส่วนประกอบสำคัญในหน้าบัญชี (Account Features)

เมื่อท่านเข้าสู่หน้าบัญชีตนเองผ่านไอคอนรูปภาพโปรไฟล์ (Avatar) หรือไอคอนผู้ใช้งานที่ส่วนบนสุด จะพบกับฟีเจอร์สำคัญดังนี้:

<Steps>
### 1. การปรับปรุงข้อมูลส่วนบุคคล (Profile Details)
สิทธิในการปรับเปลี่ยนชื่อ นามสกุล หมายเลขโทรศัพท์ และการเปลี่ยนรูปภาพประจำตัว (Avatar) เพื่อการจดจำสมาชิกที่ง่ายขึ้น

### 2. มาตรฐานความปลอดภัย (Account Security)
ส่วนจัดการเปลี่ยนรหัสผ่านใหม่ (Update Password) และการตรวจสอบที่อยู่อีเมลที่ผูกไว้กับบัญชี

### 3. การเชื่อมต่อโซเชียลมีเดีย (Social Connections)
การจัดการเชื่อมโยงบัญชี Google หรือ Discord เพื่อเพิ่มช่องทางการเข้าสู่ระบบให้หลากหลายและรวดเร็ว
</Steps>

---

## ระบบตัวแทนจำหน่ายและกุญแจ API (Reseller & API Keys)

สำหรับผู้ใช้งานระดับตัวแทนจำหน่าย (Reseller) หรือผู้ที่ต้องการบูรณาการระบบ Seluna Cloud เข้ากับแอปพลิเคชันส่วนตัว ท่านสามารถสร้างและจัดการกุญแจเข้าถึงได้ด้วยตนเอง:

<Tabs items={['API Keys', 'Reseller Status']}>
  <Tab value="API Keys">
    **วิธีการสร้างกุญแจเข้าถึง:**
    1. เข้าสู่เมนู **API Keys/กุญแจเข้าถึง**
    2. กดปุ่ม **สร้างกุญแจใหม่ (Generate New Key)**
    3. ระบุชื่อคีย์เพื่อระบุวัตถุประสงค์การใช้งาน (เช่น My-App-Integration)
    4. ระบบจะทำการสุ่มกุญแจลับ (Client Secret) ให้ท่านในทันที
    <Callout type="warning">
      **คำเตือน**: กุญแจ API จะแสดงผลเพียงครั้งเดียว โปรดบันทึกเก็บไว้ในที่ปลอดภัยสูง หากทำหายท่านต้องทำการลบและสร้างกุญแจใหม่เท่านั้น
    </Callout>
  </Tab>
  <Tab value="Reseller Status">
    **สิทธิประโยชน์เฉพาะของตัวแทน:**
    - แสดงระดับคลาส (Tier) ของตัวแทนจำหน่ายปัจจุบัน
    - อัตราส่วนลดพิเศษ (Wholesale Price) ที่จะแสดงผลเมื่อท่านทำการสั่งซื้อ
    - การตรวจสอบประวัติยอดขายและการทำกำไร (Margin) ในกรณีมีการขายต่อ
  </Tab>
</Tabs>

## การเชื่อมโยงกับ LINE เพื่อการแจ้งเตือน (LINE Integration)

ระบบได้รับการพัฒนาให้สามารถเชื่อมต่อกับ LINE Application เพื่ออำนวยความสะดวกในด้านการสื่อสาร:

- **การเริ่มใช้งาน**: คลิกที่ปุ่ม **เชื่อมต่อกับ LINE (Connect LINE Account)** และปฏิบัติตามขั้นตอนการขออนุมัติสิทธิ์ (OAuth)
- **สิทธิประโยชน์**:
  - รับข้อความแจ้งเตือนทันทีเมื่อมีการเปลี่ยนแปลงสถานะออเดอร์
  - แจ้งเตือนยอดพัสดุ และสถานการณ์ชำระเงินสำเร็จ
  - การล็อกอินเข้าสู่ระบบผ่าน LINE ได้โดยตรง (ในอนาคต)

## ขั้นตอนการส่งออกและลบข้อมูล (Data Rights)

เพื่อให้สอดคล้องกับนโยบายคุ้มครองข้อมูลส่วนบุคคล (PDPA) ท่านมีระเบียบวิธีจัดการข้อมูลดังนี้:

- **การขอดูข้อมูลทั้งหมด**: ท่านสามารถตรวจสอบประวัติการทำธุรกรรม (Transaction History) ทั้งหมดได้ตั้งแต่วันแรกที่สมัครสมาชิก
- **การขอยกเลิกบัญชี (Account Deletion)**: หากท่านต้องการปิดการใช้งานบัญชีอย่างถาวร โปรดแจ้งความจำนงผ่านเจ้าหน้าที่สนับสนุน ระบบจะทำการส่งข้อมูลสรุปคำสั่งซื้อสุดท้ายให้ท่านทางอีเมลก่อนดำเนินขั้นตอนการลบข้อมูล
- **นโยบายความเป็นส่วนตัว**: ข้อมูลของท่านจะถูกเก็บรักษาภายใต้มาตรฐานการเข้ารหัสของ Seluna Cloud และจะไม่มีการเปิดเผยต่อบุคคลภายนอกโดยไม่ได้รับอนุญาต
