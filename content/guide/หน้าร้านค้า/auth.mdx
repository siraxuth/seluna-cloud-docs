# การยืนยันตัวตนและการเข้าถึงบัญชี (Authentication)

ระบบ Seluna Cloud ให้ความสำคัญสูงสุดกับความปลอดภัยของข้อมูลผู้ใช้งาน เราจึงได้พัฒนาระบบการยืนยันตัวตนที่ทันสมัยและใช้งานง่าย เพื่อให้ท่านสามารถเข้าถึงบริการและจัดการคำสั่งซื้อได้อย่างรวดเร็วและปลอดภัยในทุกขั้นตอน

---

## ขั้นตอนการลงทะเบียนสมาชิกใหม่ (Registration)

สำหรับการใช้บริการในครั้งแรก ท่านจำเป็นต้องสร้างบัญชีผู้ใช้งานเพื่อบันทึกประวัติการสั่งซื้อและสิทธิ์การเข้าถึงผลิตภัณฑ์ โดยมีขั้นตอนดังนี้:

<Steps>
### 1. การเข้าถึงหน้าลงทะเบียน
เลือกเมนู **สมัครสมาชิก (Sign Up)** บริเวณแถบนำทางด้านบน หรือเข้าสู่ URL เส้นทาง `/auth/sign-up` โดยตรง

### 2. การระบุข้อมูลพื้นฐาน
กรอกข้อมูลส่วนบุคคลในช่องที่กำหนดให้ครบถ้วน:
- **ที่อยู่อีเมล (Email Address)**: สำหรับใช้เป็นชื่อบัญชีและรับข้อมูลแจ้งเตือน
- **รหัสผ่าน (Password)**: กำหนดรหัสผ่านที่มีความปลอดภัยสูง (แนะนำให้ประกอบด้วยอักษรตัวเล็ก ตัวใหญ่ ตัวเลข และสัญลักษณ์)
- **ยืนยันรหัสผ่าน**: ระบุรหัสผ่านเดิมอีกครั้งเพื่อความถูกต้อง

### 3. การยอมรับนโยบาย
ตรวจสอบและเลือกยอมรับข้อตกลงการใช้งาน (Terms of Service) และนโยบายความเป็นส่วนตัว (Privacy Policy)

### 4. การยืนยันการสมัคร
กดปุ่ม **สร้างบัญชี (Create Account)** เพื่อเสร็จสิ้นขั้นตอน ระบบจะทำการบันทึกข้อมูลและนำท่านเข้าสู่ระบบโดยอัตโนมัติในครั้งแรก
</Steps>

## รูปแบบการเข้าสู่ระบบ (Login Methods)

เพื่อให้สอดคล้องกับพฤติกรรมการใช้งานที่หลากหลาย ระบบรองรับช่องทางการเข้าสู่ระบบ 2 รูปแบบหลัก:

<Tabs items={['การใช้อีเมลและรหัสผ่าน', 'การใช้บัญชีโซเชียล (Social Login)']}>
  <Tab value="การใช้อีเมลและรหัสผ่าน">
    **ขั้นตอนการดำเนินงาน:**
    1. ระบุที่อยู่อีเมลที่ใช้ลงทะเบียนไว้ในช่อง **ชื่อผู้ใช้งาน/อีเมล**
    2. กรอกรหัสผ่านที่กำหนดให้ถูกต้อง
    3. เลือกตัวเลือก **จดจำฉัน (Remember Me)** หากต้องการให้ระบบคงสถานะการเข้าสู่ระบบไว้ในอุปกรณ์นี้
    4. กดปุ่ม **เข้าสู่ระบบ (Sign In)**
  </Tab>
  <Tab value="การใช้บัญชีโซเชียล (Social Login)">
    **ขั้นตอนการดำเนินงาน:**
    1. เลือกปุ่มสัญลักษณ์บริการที่ท่านต้องการเชื่อมต่อ (อาทิ **Google** หรือ **Discord**)
    2. ยืนยันสิทธิ์การเข้าถึงข้อมูลพื้นฐานผ่านหน้าต่างป๊อปอัปของบริการนั้นๆ
    3. ระบบจะทำการตรวจสอบและเชื่อมโยงบัญชี Seluna ของท่านเข้ากับบัญชีโซเชียลโดยอัตโนมัติ
    <Callout type="info">
      วิธีนี้จัดเป็นวิธีการเข้าสู่ระบบที่มีความปลอดภัยสูงและลดขั้นตอนการจดจำรหัสผ่านหลายชุด
    </Callout>
  </Tab>
</Tabs>

## การกู้คืนรหัสผ่าน (Password Recovery)

หากท่านสูญเสียสิทธิ์การเข้าถึงบัญชีเนื่องจากจำรหัสผ่านไม่ได้ โปรดดำเนินกระบวนการกู้คืนดังนี้:

1. คลิกที่ลิงก์ **ลืมรหัสผ่านหรือไม่? (Forgot Password?)** ในหน้าเข้าสู่ระบบ
2. ระบุที่อยู่อีเมลที่ผูกไว้กับบัญชี และกดปุ่ม **ส่งคำขอกู้คืน (Send Recovery Link)**
3. **ตรวจสอบอีเมล**: ระบบจะส่งข้อความอิเล็กทรอนิกส์พร้อมลิงก์กำหนดรหัสผ่านใหม่
4. คลิกที่ลิงก์ในอีเมลเพื่อเข้าสู่หน้าการกำหนดรหัสผ่านชุดใหม่ และยืนยันการเปลี่ยนแปลง

## มาตรการรักษาความปลอดภัยของบัญชี

<Callout type="warning">
  **คำแนะนำด้านความปลอดภัย**: ทางระบบ Seluna Cloud และเจ้าหน้าที่ของร้านค้า จะไม่มีนโยบายสอบถามรหัสผ่านส่วนตัวของท่านผ่านทางแชทหรือโทรศัพท์ในทุกกรณี หากพบพฤติกรรมน่าสงสัย โปรดแจ้งเจ้าหน้าที่ทันที
</Callout>

เพื่อความปลอดภัยสูงสุดของยอดเงินใน Wallet และสิทธิ์การเข้าถึงผลิตภัณฑ์ดิจิทัลของท่าน:
- โปรดออกจากระบบ (Sign Out) ทุกครั้งหลังเสร็จสิ้นการใช้งานบนอุปกรณ์สาธารณะ
- หลีกเลี่ยงการใช้รหัสผ่านที่คาดเดาได้ง่าย เช่น วันเดือนปีเกิด หรือหมายเลขโทรศัพท์
- ตรวจสอบประวัติการเข้าใช้งานล่าสุดผ่านหน้าจัดการบัญชีอยู่เสมอ
