**Wallet** คือระบบยอดเงินคงเหลือล่วงหน้า (Pre-paid Credit) ในบัญชี Seluna Cloud ของท่าน ซึ่งถูกพัฒนาขึ้นเพื่อรองรับพฤติกรรมการซื้อที่เน้นความรวดเร็วและความสะดวกสบายสูงสุด โดยเฉพาะการซื้อสินค้าดิจิทัลที่ต้องการเข้าถึงข้อมูลในทันทีโดยไม่ต้องผ่านขั้นตอนขั้นตอนชำระเงินจากภายนอกซ้ำซ้อน

---

## ขั้นตอนการส่งออกและตรวจสอบยอดเงิน (Wallet Monitoring)

ท่านสามารถเข้าถึงส่วนงานจัดการเงินสำรองได้ที่เมนู **กระเป๋าเงิน (Wallet)** หรือเข้าสู่ URL เส้นทาง `/account/wallet` โดยตรง หน้านี้จะรวบรวมข้อมูลเชิงสถิติที่สำคัญดังนี้:

- **ยอดเงินปัจจุบัน (Current Balance)**: จำนวนเครดิตสุทธิที่ท่านสามารถนำมาแลกเปลี่ยนสินค้าได้ในขณะนี้
- **ประวัติการทำรายการ (Transaction Ledger)**: บันทึกความเคลื่อนไหวทางการเงินทั้งหมด ได้แก่:
  - **รายการซื้อ (Expenditure)**: ยอดเงินที่ถูกหักออกจากการชั่งซื้อสินค้า
  - **รายการเติมเงิน (Deposits)**: ยอดเครดิตที่เพิ่มเข้ามาใหม่
  - **รายการคืนเงิน (Credits/Refunds)**: ยอดเงินที่ได้รับคืนกรณีที่มีการยกเลิกออเดอร์
- **สถานะคำขอเติมเงิน (Top-up Status)**: ติดตามความคืบหน้าของยอดเงินที่ท่านส่งเข้าสู่ระบบ (รอดำเนินการ, สำเร็จ หรือถูกปฏิเสธ)

## กลยุทธ์การใช้งาน Wallet ในการซื้อสินค้า (Spending Strategies)

ท่านสามารถเลือกใช้ยอดเงินใน Wallet ได้ 2 รูปแบบ ตามช่องทางที่ท่านสะดวก:

<Steps>
### การซื้อแบบทันที (Express Buy)
ในหน้ารวมสินค้า ท่านสามารถเลือกสินค้าดิจิทัลและใช้ปุ่ม **ซื้อด้วย Wallet (Buy with Wallet)** ระบบจะทำการหักเงินและส่งมอบสินค้าให้ท่านภายใน 1-3 วินาที (เหมาะสำหรับการซื้อคีย์เกม หรือสินค้าที่มีโปรโมชันจำกัดเวลา)

### การชำระเงินผ่าน Checkout (Mixed Payment)
ในหน้าสรุปยอดรวม (Checkout Page) ท่านสามารถเลือกวิธีการชำระเงินเป็น **Wallet** ยอดเงินทั้งหมดในออเดอร์จะถูกหักออกจากบัญชีของท่านทันที และเปลี่ยนสถานะออเดอร์เป็นจ่ายเงินครบถ้วน (Paid) โดยอัตโนมัติ
</Steps>

## ขั้นตอนการเพิ่มยอดเงินในระบบ (Top-up Procedures)

ท่านสามารถเพิ่มยอดเงินเข้าสู่ Wallet ได้ผ่าน 2 วิธีการมาตรฐาน:

<Tabs items={['การเติมผ่านคูปองเติมเงิน', 'การโอนเงินและอัปโหลดสลิป']}>
  <Tab value="การเติมผ่านคูปองเติมเงิน">
    **ขั้นตอนการดำเนินงาน:**
    1. เข้าไปที่เมนู **เติมเงิน (Top-up)**
    2. เลือกหัวข้อ **รหัสคูปอง (Prepaid Code)**
    3. ระบุรหัส 16 หลัก หรือรหัสที่ได้รับจากกิจกรรมส่งเสริมการขาย
    4. กดปุ่ม **ใช้รหัส (Redeem)**
    <Callout type="info">
      ยอดเงินจะถูกเพิ่มเข้าสู่บัญชีในทันที (Instant Balance Update) โดยไม่ต้องรอการอนุมัติ
    </Callout>
  </Tab>
  <Tab value="การโอนเงินและอัปโหลดสลิป">
    **ขั้นตอนการดำเนินงาน:**
    1. ระบุจำนวนเงินที่ต้องการเติม (เช่น 500 บาท)
    2. เลือกช่องทางโอนเงิน และดำเนินการโอนผ่านแอปธนาคาร (QR Code / PromptPay)
    3. บันทึกรูปภาพหลักฐานการโอน (Slip) และนำมาอัปโหลดเข้าสู่ระบบ
    4. กดปุ่ม **ส่งคำขอ (Submit Request)**
    <Callout type="warning">
      **การตรวจสอบ**: เจ้าหน้าที่ของร้านจะทำการตรวจสอบยอดเงินภายในระยะเวลาทำการปกติ หากข้อมูลถูกต้อง ยอดเงินจะถูกปรับปรุงในภายหลัง
    </Callout>
  </Tab>
</Tabs>

## มาตรการรักษาความปลอดภัยของยอดเงินเครดิต

<Callout type="warning">
  **คำแนะนำเชิงนโยบาย**: ยอดเงินใน Wallet ไม่สามารถแลกเปลี่ยนกลับมาเป็นเงินสดได้ในทุกกรณี (Non-refundable) โปรดเติมเงินเท่าที่จำเป็นสำหรับการสั่งซื้อสินค้าที่ท่านต้องการเท่านั้น
</Callout>

เพื่อป้องกันสิทธิ์ของท่าน:
- การโอนเงินต้องมียอดทศนิยมที่ตรงตามที่ระบบระบุ (หากมีการกำหนด)
- ห้ามทำการดัดแปลงรูปภาพสลิปการโอนเงิน เนื่องจากระบบมีเครื่องมือตรวจสอบความโปร่งใสอัตโนมัติ
- หากยอดเงินไม่เข้าภายใน 30 นาทีหลังการยืนยันสลิป โปรดแจ้งฝ่ายสนับสนุนพร้อมรูปถ่ายหลังฐานในทันที
