Seluna Cloud คือโซลูชันอีคอมเมิร์ซแบบ **Multi-tenant** ทรงประสิทธิภาพ ที่ได้รับการออกแบบมาเพื่อรองรับธุรกิจที่ต้องการความเป็นอิสระสูงสุดในการบริหารจัดการ ระบบสถาปัตยกรรมของเราช่วยให้เจ้าของร้านค้าสามารถสร้าง ปรับแต่ง และดำเนินกิจการสถานประกอบการออนไลน์ได้อย่างอิสระ โดยแต่ละร้านค้าจะได้รับการจัดสรรทรัพยากรที่แยกออกจากกันอย่างเป็นสัดส่วน (Isolation) ทั้งในส่วนของฐานข้อมูลลูกค้า รายการผลิตภัณฑ์ และการตั้งค่าอัตลักษณ์แบรนด์ ทั้งสิ้นนี้ถูกขับเคลื่อนบนโครงสร้างพื้นฐานระดับสูงร่วมกัน

---

## องค์ประกอบหลักของระบบ

แพลตฟอร์ม Seluna Cloud ประกอบด้วยสามแอปพลิเคชันหลักที่ทำงานสอดประสานกันอย่างเป็นระบบ ดังนี้

1.  **Storefront (หน้าร้านออนไลน์)** — ส่วนติดต่อกับลูกค้า (UI) ที่เน้นประสบการณ์การใช้งานที่ราบรื่น (UX) รองรับการเรียกดูผลิตภัณฑ์ การทำธุรกรรม และการบริหารจัดการบัญชีส่วนบุคคล
2.  **Portal (ศูนย์จัดการเจ้าของร้าน)** — แผงควบคุมระดับบริหารสำหรับเจ้าของธุรกิจ เพื่อใช้ในการก่อตั้งร้านค้าใหม่ กำหนดค่าโดเมนเนม ตรวจสอบสถานะการสมัครสมาชิก และจัดการนโยบายในภาพรวม
3.  **Auth App (ระบบยืนยันตัวตนกลาง)** — บริการด้านความปลอดภัยที่ทำหน้าที่เป็นศูนย์กลางการตรวจสอบตัวตน (Centralized Authentication) ให้ความปลอดภัยระดับสูงและรองรับการใช้งานข้ามระบบอย่างไร้รอยต่อ

## กลุ่มผู้ใช้งานและบทบาทหน้าที่

ระบบถูกออกแบบมาเพื่อรองรับการใช้งานของบุคคลสามกลุ่มหลัก เพื่อให้การดำเนินธุรกิจเป็นไปอย่างมีประสิทธิภาพสูงสุด

### ลูกค้าและผู้เยี่ยมชม (Customers)
กลุ่มผู้สั่งซื้อที่เข้ามามีปฏิสัมพันธ์กับหน้าร้านค้า โดยมีฟังก์ชันรองรับความต้องการดังนี้:
- **การเข้าถึงข้อมูล**: การเลือกชมผลิตภัณฑ์ทั้งประเภทสินค้าดิจิทัลและสินค้าอุปโภคบริโภคทั่วไป
- **ธุรกรรมทางการเงิน**: ระบบตะกร้าสินค้าอัจฉริยะ และการชำระเงินผ่านช่องทางที่หลากหลาย
- **การจัดการตนเอง**: การตรวจสอบสถานะการจัดส่ง การกู้คืนรหัสผลิตภัณฑ์ที่มียอดสั่งซื้อ และการบริหารยอดเงินในกระเป๋าเงินอิเล็กทรอนิกส์

### ผู้ดูแลระบบร้านค้า (Store Admins)
บุคลากรที่รับผิดชอบการดำเนินงานรายวันภายในร้านค้าแต่ละแห่ง โดยดูแลส่วนงานดังต่อไปนี้:
- **บริหารผลิตภัณฑ์**: การจัดการคลังสต็อก การลงข้อมูลรายละเอียดสินค้า และการจัดหมวดหมู่ผลิตภัณฑ์
- **บริหารคำสั่งซื้อ**: การตรวจสอบความถูกต้องของสลิปโอนเงิน การจัดเตรียมบรรจุภัณฑ์ และการติดตามการส่งมอบ
- **การตลาดและลูกค้าสัมพันธ์**: การสร้างกิจกรรมส่งเสริมการขายผ่านคูปองส่วนลด และการดูแลระบบสมาชิก

### ผู้ดูแลระดับบริหาร (Platform Owners)
เจ้าของกิจการหรือผู้บริหารที่ดูแลภาพรวมของสถานประกอบการในเครือทั้งหมด:
- **การขยายธุรกิจ**: การสร้างสถานประกอบการใหม่ (Sites Construction) ภายใต้ชื่อโดเมนที่กำหนดเอง
- **นโยบายระดับสูง**: การกำหนดค่าบริการจัดส่งพัสดุ และการเลือกใช้ผู้ให้บริการทางด้านธุรกรรมการเงิน
- **การตรวจสอบ**: การดูสถิติภาพรวมและการเข้าถึงประวัติการสื่อสารผ่านระบบอีเมล

## คุณสมบัติเด่นของแพลตฟอร์ม

| คุณสมบัติ | รายละเอียดและประโยชน์ที่ได้รับ |
| :--- | :--- |
| **Automated Digital Delivery** | ระบบจัดส่งชุดข้อมูลดิจิทัลโดยอัตโนมัติทันทีที่การชำระเงินเสร็จสมบูรณ์ พร้อมการเข้ารหัสรักษาความลับข้อมูล |
| **Logistics Integration** | บูรณาการร่วมกับผู้ให้บริการขนส่งชั้นนำ อาทิ iShip และ Pranakorn Shipping เพื่อความแม่นยำในการติดตามสถานะ |
| **Hybrid Wallet System** | ระบบกระเป๋าเงินที่รองรับทั้งการชำระเงินล่วงหน้าและการสะสมแต้มเพื่อการซื้อที่รวดเร็ว (Instant Checkout) |
| **Secure Payment Gateway** | รองรับการเชื่อมต่อโดยตรงกับ Payso และ Beam รวมถึงเทคโนโลยีป้องกันการทุจริตทางการเงิน |
| **Social Cloud Connector** | การเชื่อมโยงกับ LINE เพื่อการแจ้งเตือนสถานะทันทีผ่านมือถือ และการรวมศูนย์ข้อมูลผู้ใช้งาน |
| **Omni-CMS** | ระบบจัดการเนื้อหา (Blog) สำหรับการประชาสัมพันธ์ข่าวสารที่แสดงผลได้อย่างสวยงามบนทุกอุปกรณ์ |

## ขั้นตอนการศึกษาเอกสารคู่มือ

เพื่อให้ท่านได้รับข้อมูลที่ตรงกับความต้องการมากที่สุด โปรดดำเนินการตามคำแนะนำด้านล่าง

- **สำหรับผู้ที่ต้องการเริ่มต้นการค้า**: โปรดศึกษาจากส่วน **[ภาพรวม Portal](./จัดการร้านค้า/overview.mdx)** เพื่อทำความเข้าใจขั้นตอนการสร้างร้านแรก
- **สำหรับทีมบริหารงานขาย**: โปรดศึกษาจากส่วน **[สินค้าและหมวดหมู่](./แอดมิน/products.mdx)** เพื่อเตรียมความพร้อมของหน้าร้าน
- **สำหรับคู่ค้าและลูกค้า**: โปรดศึกษาจากส่วน **[การชำระเงิน](./หน้าร้านค้า/cart-checkout.mdx)** เพื่อความเข้าใจในขั้นตอนการทำธุรกรรม

<Callout type="note">
  เอกสารชุดนี้จะมีการปรับปรุงข้อมูลให้ทันสมัยอยู่เสมอ เพื่อรองรับฟีเจอร์ใหม่ๆ ที่จะเกิดขึ้นในอนาคต หากท่านพบบทความที่ไม่ครอบคลุมปัญหาของท่าน โปรดติดต่อศูนย์ดูแลลูกค้าผ่านช่องทางที่ระบบกำหนด
</Callout>
