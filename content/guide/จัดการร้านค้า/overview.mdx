# ภาพรวมการใช้งาน Portal

**Portal** คือ "ศูนย์กลางการจัดการ" (Control Center) สำหรับเจ้าของธุรกิจ เพื่อใช้บริหารภาพรวมของร้านค้าทั้งหมดที่มีอยู่ในระบบ โดย Portal จะแยกส่วนการทำงานออกจากหน้าเว็บร้านค้า (Storefront) และระบบจัดการหลังบ้านของแต่ละร้าน (Admin Dashboard) อย่างชัดเจน

ให้ลองจินตนาการว่า:
- **Portal** คือ "สำนักงานใหญ่" ที่คุณใช้เปิดร้านสาขาใหม่ หรือจัดการเรื่องสัญญาและค่าใช้จ่าย
- **Admin Dashboard** คือ "เคาน์เตอร์หลังร้าน" ในแต่ละสาขา ที่พนักงานใช้ลงสินค้าและรับออเดอร์

## ความแตกต่างระหว่าง Portal และ Admin Dashboard

เพื่อให้เห็นภาพชัดเจนขึ้น นี่คือข้อแตกต่างหลัก:

| หัวข้อ | Portal | Admin Dashboard |
| :--- | :--- | :--- |
| **ผู้ใช้งาน** | เจ้าของร้าน (Store Owner) | ผู้ดูแลร้าน (Admin / Staff) |
| **หน้าที่หลัก** | สร้างร้านค้า, จัดการโดเมน, จัดการสมาชิกแพ็กเกจ | เพิ่มสินค้า, จัดการออเดอร์, ดูรายงานยอดขาย |
| **ขอบเขต** | จัดการทุกร้านค้าที่คุณเป็นเจ้าของ | จัดการเฉพาะร้านค้าที่เลือกเท่านั้น |
| **เป้าหมาย** | การตั้งค่าโครงสร้างธุรกิจ | การดำเนินงานขายรายวัน |

## การเข้าใช้งาน Portal

คุณสามารถเข้าถึง Portal ได้ผ่าน URL ที่ทางระบบกำหนด (โดยปกติคือกดผ่านเมนูของแพลตฟอร์มหลัก)

### การเข้าสู่ระบบ (Login)
1. เข้าไปที่หน้าเว็บไซต์ Portal
2. เลือกช่องทางการเข้าสู่ระบบ:
   - **ใช้อีเมล**: กรอกอีเมลและรหัสผ่านที่คุณลงทะเบียนไว้
   - **Google**: เข้าสู่ระบบผ่านบัญชี Google เพื่อความสะดวกรวดเร็ว
3. เมื่อสำเร็จ ระบบจะพาคุณเข้าสู่หน้าแดชบอร์ดหลัก

### การสมัครสมาชิก (Sign Up)
หากคุณยังไม่มีบัญชี สามารถสมัครได้ง่ายๆ:
1. คลิกที่ปุ่ม **สมัครสมาชิก (Sign Up)**
2. กรอกข้อมูลพื้นฐานให้ครบถ้วน
3. ระบบจะส่งลิงก์ยืนยันตัวตนไปที่อีเมลของคุณ (โปรดตรวจสอบในกล่องจดหมายขยะ หากไม่พบในกล่องขาเข้า)
4. คลิกยืนยันและเริ่มสร้างร้านค้าแรกของคุณได้ทันที

## หน้าแดชบอร์ดหลัก

เมื่อเข้าสู่ระบบแล้ว คุณจะพบกับเมนูสำคัญดังนี้:

![หน้าแดชบอร์ด Portal](/overview/portal/home.png)

- **ร้านค้าของคุณ (Your Sites)** — แสดงรายการร้านค้าทั้งหมดที่คุณดูแล คุณสามารถคลิกเข้าไปจัดการแต่ละร้านได้จากที่นี่
- **สถานะการสมัครสมาชิก (Subscription)** — ตรวจสอบแพ็กเกจปัจจุบัน วันหมดอายุ และการชำระเงิน
- **ทางลัด (Quick Links)** — เมนูทางด่วนสำหรับการสร้างร้านค้าใหม่ หรือการเข้าถึงคู่มือช่วยเหลืออื่นๆ

---

## การเปลี่ยนภาษา

Portal รองรับการใช้งานหลายภาษา เพื่อให้คุณจัดการธุรกิจได้คล่องตัวที่สุด:
- คุณสามารถเปลี่ยนภาษาได้ที่ **เมนูเลือกภาษา (Language Selector)** บริเวณส่วนบน (Header) ของหน้าจอ
- ระบบจะจดจำภาษาที่คุณเลือกไว้สำหรับการใช้งานในครั้งถัดไป
