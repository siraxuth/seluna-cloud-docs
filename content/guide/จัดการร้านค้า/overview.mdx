---


**Portal** คือระบบ "ศูนย์กลางการควบคุม" (Central Control Center) สำหรับเจ้าของกิจการหรือผู้บริหารสูงสุด (Platform Owner) เพื่อใช้ในการบริหารจัดการภาพรวมของร้านค้าและบริการทั้งหมดภายในเครือข่าย Seluna Cloud โดยระบบ Portal จะทำงานแยกส่วนออกจากส่วนติดต่อลูกค้า (Storefront) และระบบจัดการหลังบ้านรายสาขา (Admin Dashboard) อย่างชัดเจนเพื่อความปลอดภัยสูงสุด

ลองพิจารณาความแตกต่างตามลำดับโครงสร้างดังนี้:
- **Portal** เปรียบเสมือน "สำนักงานใหญ่" ที่ดูแลเรื่องการจดทะเบียนสาขาใหม่ การบริหารจัดการสัญญา และนโยบายภาพรวม
- **Admin Dashboard** เปรียบเสมือน "เคาน์เตอร์หลังร้าน" ในแต่ละสาขา ซึ่งทีมงานใช้ลงรายการสินค้า และจัดการคำสั่งซื้อรายวัน

---

## ตารางเปรียบเทียบความแตกต่างระหว่าง Portal และ Admin Dashboard

เพื่อให้ท่านสามารถปฏิบัติงานได้ตรงตามวัตถุประสงค์ โปรดพิจารณาข้อแตกต่างดังต่อไปนี้:

| หัวข้อการพิจารณา | ระบบ Portal | ระบบ Admin Dashboard |
| :--- | :--- | :--- |
| **กลุ่มผู้ใช้งานหลัก** | เจ้าของธุรกิจ (Owner / CEO) | ผู้ดูแลร้าน (Manager / Staff) |
| **หน้าที่ความรับผิดชอบ** | การก่อตั้ง Site, การทำโดเมนเนม, สมัครแพ็กเกจ | บันทึกสินค้า, จัดการออเดอร์, ดูสรุปยอดขาย |
| **ขอบเขตการเข้าถึง** | บริหารทุก Sites ภายใต้การดูแลของเจ้าของ | บริหารได้เฉพาะ Site ที่ได้รับอนุญาตเท่านั้น |
| **วัตถุประสงค์เชิงกลยุทธ์** | การวางโครงสร้างและทิศทางธุรกิจ | การดำเนินงานพาณิชย์รายบุคคล |

## ขั้นตอนการเข้าใช้งานระบบ (Access Procedures)

ท่านสามารถเข้าถึง Portal ได้ผ่านที่อยู่ URL ที่ได้รับจัดสรรเป็นการเฉพาะจากระบบ โดยมีขั้นตอนที่สำคัญดังนี้

### การเข้าสู่ระบบ (Authentication)

<Tabs items={['การใช้อีเมล', 'การใช้บัญชี Google']}>
  <Tab value="การใช้อีเมล">
    **ขั้นตอนการดำเนินงาน:**
    1. ระบุที่อยู่อีเมลที่ท่านใช้ในการจดทะเบียนบัญชีเจ้าของธุรกิจ
    2. กรอกรหัสผ่านที่กำหนดไว้อย่างถูกต้อง
    3. กดปุ่มเพื่อยืนยันการเข้าใช้งาน ระบบจะดำเนินการตรวจสอบสิทธิ์และนำท่านเข้าสู่แดชบอร์ดหลักในทันที
  </Tab>
  <Tab value="การใช้บัญชี Google">
    **ขั้นตอนการดำเนินงาน:**
    1. เลือกปุ่มสัญลักษณ์ Google เพื่อเชื่อมต่อบัญชี
    2. ยืนยันตัวตนผ่านหน้าต่างของ Google Account
    3. เมื่อสำเร็จ ระบบจะทำการซิงค์ข้อมูลและนำท่านเข้าสู่พอร์ทัลโดยอัตโนมัติ (แนะนำวิธีนี้เพื่อความรวดเร็วและความปลอดภัยสูงสุด)
  </Tab>
</Tabs>

### การลงทะเบียนบัญชีใหม่ (Registration)

หากท่านยังไม่มีบัญชีผู้ใช้งาน โปรดดำเนินการตามขั้นตอนดังนี้:
1. เลือกเมนู **ลงทะเบียนสมาชิกใหม่ (Sign Up)** เพื่อเข้าสู่ขั้นตอนแรก
2. ระบุข้อมูลพื้นฐานตามข้อกำหนดของระบบให้ครบถ้วนทุกช่องทาง
3. **การยืนยันตัวตน**: ระบบจะส่งข้อความอิเล็กทรอนิกส์ยืนยันสถานะไปที่อีเมลของท่าน
   <Callout type="warning">
     หากท่านไม่พบข้อความในกล่องจดหมายหลัก (Inbox) โปรดตรวจสอบในส่วนของจดหมายขยะ (Spam / Junk) เพื่อดำเนินขั้นตอนให้เสร็จสมบูรณ์
   </Callout>
4. คลิกที่ลิงก์ที่ได้รับเพื่อเปิดใช้งานบัญชี และเริ่มก่อตั้งร้านค้าแรกของท่านได้ทันที

## ส่วนประกอบสำคัญบนแดชบอร์ดหลัก

เมื่อท่านเข้าสู่ระบบสำเร็จ จะพบกับเมนูหลักที่ใช้ในการวิเคราะห์และจัดการดังนี้

<Steps>
### การจัดการร้านค้า (Your Sites)
แสดงรายการสถานประกอบการทั้งหมดภายใต้การดูแล ท่านสามารถตรวจสอบสถานะออนไลน์ของแต่ละร้าน และเข้าสู่เมนูตั้งค่าเฉพาะรายสาขาได้จากหน้านี้

### การบริหารการสมัครสมาชิก (Subscription Management)
ตรวจสอบรายละเอียดรอบการชำระเงินประเภทแพ็กเกจที่ใช้งานอยู่ปัจจุบัน รวมถึงวันที่หมดอายุและการขอรายงานสรุปยอดค่าบริการ

### ทางลัดและสถิติ (Quick Links & Analytics)
เมนูทางด่วนสำหรับการก่อตั้งร้านค้าใหม่แบบเร่งด่วน และส่วนของการเข้าถึงเอกสารคู่มือเชิงเทคนิคฉบับย่อ
</Steps>

## นโยบายการรองรับภาษา (Multilingual Support)

ระบบ Portal ได้รับการพัฒนาให้รองรับการแสดงผลหลากหลายภาษา เพื่ออำนวยความสะดวกในการบริหารธุรกิจในระดับสากล

- ท่านสามารถปรับเปลี่ยนภาษาที่ต้องการใช้งานได้ผ่าน **ตัวเลือกภาษา (Language Selector)** บริเวณแถบเมนูด้านบนของหน้าจอ
- ระบบจะทำการจดจำค่าที่คุณเลือกไว้โดยอัตโนมัติ (Session Persistence) เพื่อให้การเข้าใช้งานในครั้งถัดไปแสดงผลด้วยภาษาเดิมทันที
- การเปลี่ยนภาษาใน Portal จะไม่มีผลกระทบต่อภาษาที่ใช้แสดงผลบนหน้าร้านค้า (Storefront) ของลูกค้า
