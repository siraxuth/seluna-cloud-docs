# ภาพรวม Portal

Portal คือแอปพลิเคชันสำหรับเจ้าของร้านค้าในการสร้างและจัดการร้านค้าออนไลน์บนแพลตฟอร์ม Portal แยกต่างหากจาก storefront และ admin dashboard — มีไว้สำหรับจัดการโครงสร้างพื้นฐานของร้านค้า ไม่ใช่การดำเนินงานประจำวัน

## ความแตกต่างระหว่าง Portal กับ Admin Dashboard

| Portal | Admin Dashboard |
|---|---|
| สำหรับเจ้าของร้านค้า (store owner) | สำหรับผู้ดูแลระบบร้านค้า (store admin) |
| สร้างและตั้งค่าร้านค้า | จัดการสินค้า ออเดอร์ และลูกค้า |
| จัดการแผนการสมัครสมาชิก | ดำเนินงานประจำวัน |
| ควบคุมระดับแพลตฟอร์ม | ควบคุมระดับร้านค้าเดียว |

## การเข้าถึง Portal

Portal อยู่ที่ `apps/portal` และทำงานบน port 3000 แยกต่างหากจาก storefront

### การเข้าสู่ระบบ Portal

1. เข้าไปที่ URL ของ Portal
2. คลิก **เข้าสู่ระบบ**
3. กรอกอีเมลและรหัสผ่านของบัญชี Portal
4. หรือเลือก **Google** เพื่อเข้าสู่ระบบด้วยบัญชี Google
5. คุณจะถูกพาไปยังแดชบอร์ด Portal

### การสมัครสมาชิก Portal

1. คลิก **สมัครสมาชิก**
2. กรอกอีเมลและรหัสผ่าน
3. ยืนยันอีเมลผ่านลิงก์ที่ส่งไป
4. หลังยืนยัน เข้าสู่ระบบและเริ่มสร้างร้านค้าได้

### ลืมรหัสผ่าน

1. คลิก **ลืมรหัสผ่าน**
2. กรอกอีเมล
3. รับลิงก์รีเซ็ตรหัสผ่านทางอีเมล
4. คลิกลิงก์และตั้งรหัสผ่านใหม่

## หน้าหลัก Portal

หลังจากเข้าสู่ระบบ คุณจะเห็นแดชบอร์ดหลักที่แสดง:

![หน้าแดชบอร์ด Portal](/overview/home.png)

- **ร้านค้าของคุณ** — รายการ site/ร้านค้าทั้งหมดที่คุณเป็นเจ้าของ
- **สถานะการสมัครสมาชิก** — แผนที่คุณใช้อยู่
- **ลิงก์ด่วน** — ไปยังการสร้างร้านค้าใหม่ การตั้งค่า และเอกสาร

---

## การรองรับหลายภาษา

Portal รองรับหลายภาษา โดย URL มีรูปแบบ `/[locale]/portal/...` เช่น:
- `/th/portal/` — ภาษาไทย
- `/en/portal/` — ภาษาอังกฤษ

เลือกภาษาได้จากตัวเลือกภาษาในส่วนหัวของ Portal
