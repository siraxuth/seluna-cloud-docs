# การกำหนดค่าพื้นฐานและอัตลักษณ์แบรนด์ (Site Settings & Branding)

ความสำเร็จของร้านค้าออนไลน์ Seluna Cloud เริ่มต้นจากการสร้างความประทับใจแรกเห็น (First Impression) ผ่านการออกแบบที่สวยงามและใช้งานง่าย เมนูการตั้งค่า (Settings) จะช่วยให้ผู้ดูแลระบบสามารถควบคุมภาพลักษณ์และฟังก์ชันการทำงานหลักของร้านได้อย่างสมบูรณ์แบบ

---

## ขั้นตอนการส่งออกและจัดการอัตลักษณ์แบรนด์ (Branding & Identity)

ท่านสามารถปรับแต่งภาพลักษณ์ของแบรนด์ได้ที่เมนู **การตั้งค่า → ปรับแต่งเว็บไซต์ (Branding)** หรือเส้นทาง `/backend/settings/branding`:

### 1. การกำหนดภาพสัญลักษณ์ (Visual Assets)
- **โลโก้หลัก (Primary Logo)**: ภาพตราสัญลักษณ์ที่จะปรากฏบนแถบนำทาง (Navbar) และส่วนสรุปออเดอร์
- **ไอคอนเว็บไซต์ (Favicon)**: รูปภาพขนาดเล็กที่แสดงบนแถบเบราว์เซอร์
- **ภาพหน้าปก (Cover Image)**: ภาพพื้นหลังสำหรับหน้าแรก หรือส่วนต้อนรับลูกค้า

### 2. การควบคุมชุดสีและรูปแบบ (Theming)
เลือก **ชุดสีประจำร้าน (Brand Palette)** ที่สะท้อนบุคลิกธุรกิจ:
- **สีหลัก (Primary Color)**: ใช้กับปุ่มคำสั่งหลัก ลิงก์ และจุดเน้นสำคัญ
- **สีพื้นหลัง (Background)**: กำหนดโทนสว่างหรือโทนมืด (Light/Dark Mode) ให้เหมาะสม
- **รูปแบบฟอนต์ (Typography)**: เลือกใช้ชุดตัวอักษรที่อ่านง่ายและดูเป็นสากล

    2. **Promotion Banners**: อัปโหลดภาพกิจกรรมส่งเสริมการขายรายสัปดาห์

## การตั้งค่าช่องทางรับชำระเงินและธุรกรรม (Payment Gateway Settings)

<Callout type="info">
  **คำแนะนำด้านการเงิน**: โปรดตรวจสอบความถูกต้องของหมายเลขบัญชีธนาคารและ API Keys ของผู้ให้บริการชำระเงินอย่างถี่ถ้วน เพื่อป้องกันปัญหาการรับเงินผิดพลาด
</Callout>

**พิจารณากำหนดค่าส่วนงานดังนี้:**
- **ธนาคาร (Bank Accounts)**: สำหรับแสดงเลขบัญชีให้ลูกค้าโอนเงินและอัปโหลดสลิป
- **เกตเวย์อัตโนมัติ (Automatic Gateways)**: เชื่อมต่อบัญชี Payso, Beam หรือช่องทางอื่นเพื่อการอนุมัติออเดอร์ทันใจ
- **สกุลเงินและภาษี (Currency & Tax)**: กำหนดสกุลเงินที่ใช้จำหน่าย และสัดส่วนภาษีมูลค่าเพิ่ม (VAT) ในกรณีที่ต้องการแสดงราคาแยกรายการ

---

## ข้อมูลพื้นฐานและการตั้งค่าโครงสร้างระบบ (General Configuration)

- **ข้อมูลนโยบายสถาบัน (Basic Info)**: ชื่อร้านค้า ที่อยู่ออฟฟิศ อีเมล และเบอร์โทรศัพท์ประจำสาขา
- **สถานะการทำงาน (Operational Status)**:
  - **เปิดให้บริการ (Open)**: ร้านค้าดำเนินกิจกรรมพาณิชย์ตามปกติ
  - **ปิดเพื่อปรับปรุง (Maintenance Mode)**: ปิดหน้าบ้านชั่วคราวเพื่ออัปเดตข้อมูลขนาดใหญ่
- **โดเมนเนม (Domain Management)**: การตั้งค่าที่อยู่ URL ล่าสุด และการตรวจสอบใบรับรองความปลอดภัย HTTPS

---

## ระเบียบการรักษาความปลอดภัยของระบบตั้งค่า

<Callout type="warning">
  **คำแนะนำระดับบริหารสูงสุด**: การเปลี่ยนแปลงข้อมูลในส่วนการตั้งค่าอาจส่งผลต่อการมองเห็นของลูกค้าทั่วแพลตฟอร์ม โปรดตรวจสอบความเรียบร้อย (Preview) ทุกครั้งก่อนทำการบันทึกข้อมูลอย่างเป็นทางการ
</Callout>

ขอบเขตการทำงานของทีมงานในส่วนนี้:
- ทีมแอดมินสามารถกำหนดสิทธิ์ (Permission Level) ให้พนักงานคนอื่นเข้าถึงเฉพาะบางส่วนของการตั้งค่าได้
- การเปลี่ยนแปลงในส่วน Gateway จะถูกบันทึกประวัติ (Audit Log) อย่างละเอียดเพื่อความปลอดภัย
- หากพบปัญหาในการแสดงผลหลังปรับแต่งธีมใหม่ โปรดติดต่อฝ่ายเทคนิคเพื่อตรวจสอบโครงสร้าง CSS ทันที
